# telnet 命令

开启 Windows telent 服务

点击键盘 windows 键，选择 设置 ，在 Windows Settings 页面搜索 telnet , 勾选 Telnet Client 服务，点击 OK .

打开 windows 上的 cmd 或 powershell 执行如下命令

	// 测试本机是否能与服务器某个端口连通
	$ telnet 10.67.108.211 22
