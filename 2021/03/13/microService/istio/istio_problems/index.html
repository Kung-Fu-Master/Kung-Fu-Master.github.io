# ingressgateway 部署出错

部署完 Istio 后发现 ingressgateway 等 READY 一直处于 0/1 状态.

原因是 DNS域名解析不了 ，请参考上面解决 DNS 域名解析不了的解决方案.

# pod 无法创建

有时候 deploy 服务时候发现没有 POD, 如果用 deployment 部署，查看 deployment 状态没有任何报错信息，describe 查看 deployment 的 Events 发现只有到分配 replicas 这一步，没有后续信息.

上面愿意还是因为 Kubernetes 服务集群 DNS 解析有问题，请参考上面解决 DNS 域名解析不了的解决方案.
