# 机器重启 IP 改变重新加入集群

机器重启后重新加入集群

  • 修改 /etc/hosts 里的机器 IP
  • 修改 /etc/systemd/system/docker.service.d/no-proxy.conf 或 proxy.conf 里的机器 IP
  • 修改～/.bashrc 里的 export NO_PROXY=*** 对应的机器 IP
  • 修改 /etc/kubernetes/manifests/kube-apiserver.yaml 的机器 IP
  • 修改 /etc/kubernetes/manifests/etcd.yaml 的机器 IP
