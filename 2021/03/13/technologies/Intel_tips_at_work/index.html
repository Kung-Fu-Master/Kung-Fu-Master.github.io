# 关于 cube 主机连接网络问题

cube 台式机网线连接 C 口，宿主主机安装 Virtualbox 后再安装 linux 虚拟机，网络选择桥接模式，跟宿主主机网络处于同一网段，与宿主主机互不影响.
台式机连接 A 口，宿主主机设置 proxy 后可以连接外网，但是 virtualbox 安装的虚拟机就连不到外网了
安装好虚拟机 OS，
$ sudo passwd root
输入 user 用户密码，再设置 root 密码，下次就可以 $ su root 获取 root 权限了
